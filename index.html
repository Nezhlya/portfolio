World clock Application

The Time Zone Clock Application is a web-based utility designed to help users easily track time zones around the world. It provides a simple and intuitive interface for users to view the current time in the suggested time zones list, making it particularly useful for individuals who work or communicate with people in different parts of the world.

Technologies Used:

  • HTML: Used for creating the structure and layout of the web page, including the time zone clock containers.
  • CSS: Utilized for styling and customization, including clock appearance, colors and fonts.
  • JavaScript: The core functionality of the application is powered by JavaScript, including Moment.js, Ajax and cdnjs.

React Weather Forecast

React Weather Forecast is a user-friendly web application designed to provide accurate weather predictions for locations around the world. With a clean and intuitive interface, users can access essential weather information quickly and efficiently, making it an indispensable tool for planning daily activities, travel, and outdoor adventures.

Technologies Used:

  • React: The application's frontend is built using the React JavaScript library, enabling efficient and dynamic rendering of weather data.
  • API Integration: ReactWeather Forecast integrates with weather APIs (OpenWeatherMap) to fetch and display real-time weather data.
  • CSS: Styling and layout are handled with CSS and Bootstrap for a modern and responsive design.
